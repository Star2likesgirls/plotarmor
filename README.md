#   PLOT ARMOR
Have you ever watched SMP videos such as unstable and wondered how you too could be immortal? This plugin is for you! By giving a player "Plot Armor" any lethal damage will leave them at minimum 0.5 hearts

Command Alias What it does
/plotarmor add <player> /pa add <player> Grant plot armor to a player
/plotarmor remove <player> /pa remove <player> Revoke plot armor from a player
/plotarmor list /pa list View all protected players (green = online, gray = offline)
